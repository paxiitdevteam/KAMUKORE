## Image Specifications

### Recommended Dimensions:
- **Product Card**: 600x600px (1:1 square) or 600x800px (3:4 portrait)
- **Product Detail**: 1200x1200px (1:1 square) or 1200x1600px (3:4 portrait)
- **Collection Card**: 800x600px (4:3 landscape)

### File Format:
- **Primary**: WebP
- **Fallback**: JPG/JPEG
- **Quality**: 85-90%

### File Size:
- **Card Images**: < 150KB
- **Detail Images**: < 400KB
